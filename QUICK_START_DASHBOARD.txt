📋 API ENDPOINTS
═══════════════════════════════════════════════════════════════

Auth:
  POST   /api/auth/login          - Login
  GET    /api/auth/verify         - Verify token
  GET    /api/auth/profile        - Get profile

Users (Admin):
  GET    /api/users               - List all users
  GET    /api/users/:id           - Get user details
  POST   /api/users               - Create user
  PUT    /api/users/:id           - Update user
  DELETE /api/users/:id           - Delete user
  GET    /api/users/stats/dashboard - Admin stats

Calls (existing):
  GET    /api/calls               - Get calls
  GET    /api/calls/active        - Active calls
  POST   /api/calls/:id/hangup    - Hangup call

Revenue (existing):
  GET    /api/revenue/today       - Today's revenue
  GET    /api/revenue/hourly      - Hourly breakdown
